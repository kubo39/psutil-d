# psutil for D

A system monitoring library for D, heavily inspired by [psutil](https://github.com/giampaolo/psutil). Currently support Linux only.

Currently implemented:

- cpu.stats
- memory.memory
- memory.swap
- sensors.fan
- sensors.temperature

## Examples

### Cpu

```console
$ dub examples/cpu.d
```

### Memory

```console
$ dub examples/memory.d
```

### Sensors

```console
$ dub examples/sensors.d
```
